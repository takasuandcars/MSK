概要
物流管理システム
https://merucha.com
トラックの集荷依頼、作業量及びスタッフの勤怠の管理ができます。

・集荷依頼（車種/台数等）
・クーリエ入荷予定入力
・作業量の可視化
・勤怠管理
・チャット


使用技術
・Ruby 2.6.3
・Rails 6.0.3.7
・MySQL 5.2.4
・Nginx
・Unicorn
・AWS
　・VPC
　・EC2
　・Route53
・Fullcalender
・jsQR
・Prawn
・ChartJS


機能一覧
・ユーザー登録／ログイン機能
・集荷依頼入力
・集荷依頼PDF出力
・クーリエデーター入力
・勤怠管理
（ユーザーのQRコードを読み込み、出退勤）
・チャット機能（ActionCable）
・検索機能
